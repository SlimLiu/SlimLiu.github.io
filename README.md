# SlimLiu.github.io

Personal website for 刘盎希 (Angxi Liu), hosted with GitHub Pages.

## Structure

- `/` — bilingual personal homepage
- `/blog/` — literature, AI notes, and essays
- `/projects/cs180/` — UC Berkeley CS 180 course archive
- `/CS180/` — compatibility redirects for the previous project URLs

The homepage uses plain HTML, CSS, and JavaScript. It follows the visitor's
system color scheme and remembers their Chinese/English language preference.

Blog posts live in `_posts/` and can be edited through
[Pages CMS](https://app.pagescms.org/). Uploaded article images are stored in
`assets/blog/`. GoatCounter can be enabled by adding the site's code to
`goatcounter_code` in `_config.yml`.
